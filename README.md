# Simple-HR-System

## Human Resources Management System

A simple web-based Human Resources Management System designed to help manage employees efficiently. It provides a user-friendly Arabic interface that allows adding, viewing, searching, and managing employee records.

## Requirements

- Modern web browser (Chrome, Firefox, etc.)
- Internet connection (if using remote database)
